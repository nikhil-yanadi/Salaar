A high-quality cinematic scroll animation experience inspired by the world of SALAAR.
Built completely using pure HTML, CSS, and JavaScript with smooth frame-by-frame rendering and immersive visual effects.

---

## ✨ Features

* Cinematic scroll-driven animation
* Smooth frame-by-frame canvas rendering
* Dual parallax movie sequences
* Dynamic text reveal effects
* Dark immersive UI
* Responsive design for desktop and mobile
* Glassmorphism movie information card
* Optimized image rendering performance
* Pure frontend project — no frameworks used

---

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript
* Canvas API
